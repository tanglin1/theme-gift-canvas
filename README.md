# 主题礼品创意画布生成器

[English](./README_EN.md) | 中文

将节日/活动/人物/场景主题转化为高信息密度的礼品创意画布。

## ✨ 特性

- 🎁 **主题解构** - 从节日、场景、人物中提取核心关键词
- 📦 **品类规划** - 按礼品属性拆分为实用/文创/美食/体验/定制5大类
- 💰 **价格梯度** - 覆盖入门/主流/高端/顶配四档
- 👥 **场景匹配** - 匹配送礼对象（长辈/伴侣/同事/客户/孩子）
- 📱 **传播友好** - 适合小红书/公众号/电商详情页

## 📋 输出模块

一张画布包含以下核心模块：

1. **礼品定位** - 主题核心关键词、情感诉求
2. **品类矩阵** - 5大品类推荐
3. **价格梯度** - 4档价格区间推荐
4. **适用对象** - 不同人群推荐
5. **推荐理由** - 每个推荐含名称+价格+理由

## 🚀 使用方法

本技能需要配合 OpenClaw Agent 使用。

1. 安装 OpenClaw
2. 将本技能添加到 skills 目录
3. 启动 Agent 并激活技能
4. 按照引导提供主题、预算、受众等信息
